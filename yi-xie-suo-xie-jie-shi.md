# 一些缩写解释

* REPL
  * Read–eval–print loop
  * 读取-求值-输出 循环
* cli
  * command line interface
  * 命令行接口
* preset
  * 预设
* rc
  * run control
  * 运行控制（运行时控制文件）
  * 例如：.babelrc
* Unexpected token
  * 不是被期望的标识（意外 标记）
* handle
  * 处理

| 缩 写     | 全 称          | 中文  |
| ------- | ------------ | --- |
| src     | Source       | 源代码 |
| **std** | **Standard** | 标准  |
| stg     | Storage      | 存储  |
| stm     | Stream       | 流   |
